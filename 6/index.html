ANDRHI MULYA BAYUATMADJA 1922500176 SI5J Pada pertemuan keenam diajarkan apa itu plugin dan bagaimana cara menginstal plugin pada wordpress. sebenarnya menginstal plugin sendiri cara nya sama dengan menginstal tema yaitu ada 3 cara, online, semi online, dan juga offline. Untuk menginstl online dapat diinstal secara langsug dengan mencari plugin. semi online sendiri kita mengisntalplugin dengan cara mencari plugin yang kita inginkan semisal kan plugin untuk galeri, maka akan muncul beberapa plugin untuk galeri. Selanjutnya plugin ini nantinya akan kita download, dan setelah kita download maka dapat kita instal melalui wordpres. Dan untuk menginstal offline sendiri sama dengan menginstal tema pada sebelumnya kita download terlebih dahulu plugin yang kita inginkan lalu kita ekstrak dan hasil ekstrak kita pindahkan pada menu plugin pada folder file kita. Plugin yang dipelajari pada pertemuan enam ini adalah plugin untuk galeri yang artinya kita menambahkan plugin agar memunculkan galeri pada web wordpress yang kita buta. Plugin ini pun beragam , sesuai dengan yang kita butuhkan dan kita inginkan, baik dari yang gratis hingga berbayar. Setelah itu diajarkan juga agar web yang kita buat dapat ditemukan pada wordpress dan terakhir dijelaskan sedikit mengenai uts yang akan dikerjakan pada pertemuan berikutnya.
